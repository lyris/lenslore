# App Icons

Place your app icons here:
- `icon-192.png` - 192x192px PNG icon
- `icon-512.png` - 512x512px PNG icon

You can generate icons using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

For now, you can use any 192x192 and 512x512 PNG images as placeholders.
